# Metodologia

## Ferramentas de Desenvolvimento
Para o desenvolvimento do **OnCine**, foram utilizadas as seguintes ferramentas e tecnologias:

* **Editor de Código:** Visual Studio Code.
* **Linguagens:** HTML5 (Semântico), CSS3 (Variáveis, Flexbox, Grid), JavaScript (ES6+).
* **Versionamento:** Git e GitHub.
* **API:** The Movie Database (TMDB) API.
* **Design:** Estilo "Glassmorphism" e "Neon" implementados via CSS puro.

## Gerenciamento do Projeto
A metodologia de desenvolvimento seguiu práticas ágeis baseadas no SCRUM, adaptadas para um projeto individual:
1.  **Planejamento:** Definição dos requisitos baseados na API disponível.
2.  **Execução:** Codificação modular (Estrutura -> Estilo -> Lógica).
3.  **Testes:** Verificação contínua das funcionalidades no navegador.
