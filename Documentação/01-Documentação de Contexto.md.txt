# Documentação de Contexto

## Introdução
[cite_start]O projeto **OnCine** consiste no desenvolvimento de uma aplicação web *Front-End* responsiva, focada na experiência do usuário para a exploração de catálogos cinematográficos[cite: 31, 32]. A aplicação atua como uma interface moderna e intuitiva para o consumo de dados externos, centralizando informações sobre filmes, classificações e lançamentos em um único ambiente visualmente agradável, utilizando o tema "Dark/Neon".

## Problema
[cite_start]Atualmente, os amantes de cinema enfrentam a dispersão de informações em diversas plataformas de streaming ou bases de dados complexas e pouco visuais[cite: 33, 34]. Além disso, no contexto de desenvolvimento de software, há a necessidade constante de demonstrar a capacidade técnica de integração entre interfaces modernas e APIs (Application Programming Interfaces) robustas. O problema abordado é a criação de um portfólio que resolva a necessidade de busca rápida e visualização de detalhes de filmes.

## Objetivos
[cite_start]O objetivo geral é desenvolver uma aplicação web *Front-End* que consuma a API do TMDB (The Movie Database) para exibir, filtrar e detalhar informações sobre filmes[cite: 38, 39].

**Objetivos Específicos:**
* Implementar o consumo de API RESTful utilizando `fetch` e `async/await`.
* Criar uma interface responsiva e interativa com carrossel infinito e modais de detalhes.
* Desenvolver funcionalidades de pesquisa em tempo real e filtros por categoria.
* Implementar paginação dinâmica e funcionalidades de descoberta aleatória.

## Justificativa
[cite_start]A escolha deste projeto justifica-se pela alta demanda do mercado por desenvolvedores capazes de integrar *Front-End* com serviços externos[cite: 41, 42]. O uso da API do TMDB permite simular um ambiente real de produção. Além disso, a aplicação oferece uma solução estética e funcional para usuários que desejam decidir o que assistir sem navegar por interfaces lentas, centralizando a descoberta de conteúdo.

## Público-alvo
[cite_start]O público-alvo é composto por cinéfilos que buscam informações rápidas, recrutadores técnicos avaliando portfólio e usuários casuais em busca de entretenimento[cite: 47, 48].
